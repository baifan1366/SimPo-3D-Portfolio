Three.js Ship Portfolio

This project is a creative exploration of Three.js, focusing on creating an interactive 3D ship model as part of a portfolio website. It demonstrates the power of WebGL technology in creating immersive web experiences.

The project showcases various Three.js techniques, including 3D model loading, custom shaders for water effects, and interactive camera controls. Users can explore the ship from different angles, zoom in to see details, and interact with specific parts of the model.
